# monotony
CLI Game

Monotony é um jogo de plataforma Linux, execultado diretamente no terminal, 


# Como Execultar

É necessario ter o interpretador python3 instalado

# apt update
# apt install python3
# git clone https://github.com/OdinSCRT/monotony.git
# cd monotony
# python3 main.py ou ./main.py
