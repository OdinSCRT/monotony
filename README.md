<h1 id="install">Como instalar</h1>

<p>É necessario ter o interpretador python3 instalado</p>

```
# apt update
```
```
# apt install python3
```
```
$ git clone https://github.com/OdinSCRT/monotony.git
```
<h1 id="exec">Como execultar</h1>

```
$ cd monotony
```

```
$ python3 main.py
```
